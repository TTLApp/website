## Building and running on localhost

Bundler Parcel.js has been used for the compilation (https://parceljs.org/).

First install dependencies:

```sh
npm install
```

To run in hot module reloading mode:

```sh
npm start
```

To create a production build:

```sh
npm run build-prod
```

## Running

```sh
node dist/bundle.js
```
